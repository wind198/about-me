Uptime monitoring system

A system that monitors the uptime of a website and sends alerts when it goes down.

NestJS Server
The central web API that handles all the requests from the clients as well as central orchestration of the monitoring system.
Admin dashboard
A dashboard for the admin to view the status of the monitoring system and to manage the users and the monitors.
Mondo Database
A central database to store the data of the monitoring system.
Redis
Central state storage for the whole microservice system.
Check scheduler
A cron job that create check jobs and orchestrate them to the queue.
Check agents
Workers hosted on cloud at multiple locations that runs the checks for the monitors.
Result processor
A worker that processes the results of the checks, fire outage alerts and stores check results in the database.
Data aggregator
A worker that aggregates the raw check results to user friendly metrics.
Grafana & Prometheus
Monitor system health and metrics.
